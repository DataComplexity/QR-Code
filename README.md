# QR Code Generator

A streamlined utility for generating high-quality QR codes from text, URLs, or any other data strings.

## Features

- **Instant Generation**: Real-time QR code creation.
- **Download Support**: Easily save generated codes for use.
- **Versatile Input**: Supports various data formats including URLs and plain text.
- **Modern Interface**: Features a clean, utility-focused design.

## Tech Stack

- **Core**: React, Vite
- **Library**: `qrcode.react` or equivalent
- **Styling**: CSS Flexbox/Grid
